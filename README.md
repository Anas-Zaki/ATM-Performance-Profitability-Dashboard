# ATM-Performance-Profitability-Dashboard
This project provides a comprehensive analysis of ATM operations, transactions, revenue, costs, and uptime across multiple states. Using Python for data cleaning, EDA, and SQL queries, and Power BI for interactive visualizations, it delivers actionable insights for business decisions.
